# PLC
PLC Programmable Logic Controller Software
Written in Python using Pygame-CE

### New Feature Ideas
- Save and Load files
- Connect to ESP32 IO nodes for physical connections
- More Logic Gates
- Non-boolean values, numbers, compare operations
- Number variables
